# 🔮 Mentalista - Jogo de Adivinhação

Um projeto de jogo simples, interativo e moderno, onde o computador escolhe um número secreto aleatório entre 1 e 100, e o jogador tem 7 tentativas para adivinhar o valor correto, recebendo dicas a cada tentativa.

O projeto foi desenvolvido para consolidar conceitos fundamentais de lógica e interatividade na web.

## 🚀 Tecnologias Utilizadas

- **HTML5:** Estruturação da interface do jogo.
- **CSS3:** Estilização moderna com efeito *Glassmorphism* (fundo estilo vidro) e degradê dinâmico.
- **JavaScript Vanilla:** Inteligência do jogo (geração de números, contagem de jogadas e manipulação do DOM).

## 🧠 Conceitos Praticados e Aprendidos

Este repositório marca o meu retorno aos estudos práticos e foca intensamente em:
- **Controle de Estado:** Gerenciar o número de tentativas restantes e alterar as condições do jogo dinamicamente.
- **Estruturas Condicionais Complexas:** Uso de blocos `if`, `else if` e `else` para guiar o jogador com dicas de "maior" ou "menor".
- **Objeto Math:** Utilização de `Math.random()` e `Math.floor()` para a geração de números pseudo-aleatórios em um intervalo específico.
- **Manipulação de Classes no CSS:** Esconder e exibir elementos da tela (como o botão de reiniciar) de acordo com o andamento do jogo.
